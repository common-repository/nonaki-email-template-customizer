Nonaki Email Template Customizer is a WordPress plugin designed to make customizing email templates easy and intuitive. With this plugin, you can create and modify email templates for various WordPress functionalities, such as default email layout, new user registration, password reset, and More.

Email Template Customizer offers a range of pre-designed templates, each of which can be fully customized to fit the userâ€™s brand and preferences.

Nonaki comes with a drag and drop email template builder that enables users to create professional-looking email templates without any coding skills.

With the newsletter functionality, users can create and send newsletters to their subscribers, complete with customizable templates, subscriber management.

== FEATURES INCLUDE ==

- Drag and drop template builder.
- Adding and customizing Contacts. 
- Send email to your contacts.
- Customize default WordPress password reset email template. 
- Customize default WordPress new user notification email template .
- Customize default WordPress user email change email template. 
- Customize default WordPress password change email template.
- Customize Contact form 7 email templates.
- Make email templates and send them to any email address. 
- Preview and quick preview option.
- Included few beautiful email templates presets for you.
- Individual editor elements for templates.
